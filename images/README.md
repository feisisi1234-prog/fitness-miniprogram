# 图标说明

由于微信小程序需要实际的图标文件，您需要自行添加以下图标文件到 images 文件夹中：

1. home.png - 首页图标（未选中状态）
2. home-active.png - 首页图标（选中状态）
3. training.png - 训练图标（未选中状态）
4. training-active.png - 训练图标（选中状态）
5. timer.png - 计时图标（未选中状态）
6. timer-active.png - 计时图标（选中状态）
7. profile.png - 个人中心图标（未选中状态）
8. profile-active.png - 个人中心图标（选中状态）

图标尺寸建议：81px × 81px
