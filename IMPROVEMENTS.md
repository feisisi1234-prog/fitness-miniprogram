### 3. 🛠️ 统一工具库（Utils）

**新增文件：**
- `utils/storage.js` - 统一存储管理
- `utils/stats.js` - 统计数据计算
- `utils/achievement.js` - 成就系统管理

#### storage.js - 存储管理器
```javascript
- getTrainingRecords() - 获取训练记录
- saveTrainingRecord() - 保存训练记录（自动清理90天前数据）
- getUserInfo() - 获取用户信息
- saveUserInfo() - 保存用户信息
- getAchievements() - 获取已解锁成就
- unlockAchievement() - 解锁成就
- getBodyRecords() - 获取身体数据
- saveBodyRecord() - 保存身体数据
- getFoodRecords() - 获取饮食记录
- saveFoodRecord() - 保存饮食记录
```

#### stats.js - 统计计算器
```javascript
- calculateUserStats() - 计算用户统计数据
- calculateContinuousDays() - 计算连续训练天数
- calculateTodayProgress() - 计算今日进度
- calculateWeekStats() - 计算本周统计
- generateHeatmapData() - 生成训练热力图数据
- generateMonthlyReport() - 生成月度报告
```

## 🚀 使用方式

### 查看成就
1. 进入"我的"页面
2. 点击"成就中心"
3. 查看已解锁和未解锁的成就
4. 点击成就卡片查看详情

### 记录身体数据
1. 进入"我的"页面
2. 点击"身体数据"
3. 点击右上角"+ 添加记录"
4. 填写数据并保存
5. 切换不同数据类型查看趋势

### 自动解锁成就
- 完成训练后自动检测
- 符合条件时弹窗提示
- 可在成就中心查看所有成就
