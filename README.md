# 健身训练小程序

一个功能丰富的微信小程序，帮助用户进行科学的健身训练和饮食管理。

## 功能特点

### 🏋️ 训练功能
- **多样化训练计划**：力量训练、HIIT、瑜伽、普拉提等多种训练类型
- **自定义训练计划**：用户可以创建和管理自己的训练计划
- **训练记录追踪**：记录每次训练的时长、消耗卡路里等数据
- **训练计时器**：支持多种运动类型的计时功能
- **肌肉分析**：详细的肌肉纤维走向说明和肌肉张力计算

### 🍎 饮食功能
- **食物识别**：AI智能识别食物（集成扣子AI）
- **营养分析**：自动分析食物的营养成分和卡路里

### 📊 数据统计
- **每日目标**：设置和追踪每日运动目标
- **本周统计**：查看本周训练天数、总时长、消耗卡路里等
- **连续训练天数**：激励用户保持训练习惯

### 👤 个人中心
- **用户信息管理**：编辑个人资料和头像
- **训练记录查看**：查看历史训练记录
- **设置中心**：自定义每日目标和其他设置

## 技术特点

- **SVG图标**：使用轻量级SVG图标替代传统图片，提升加载速度
- **OCR识别**：支持扫描训练计划图片，自动识别训练动作
- **本地存储**：使用微信小程序本地存储管理用户数据
- **响应式设计**：适配不同尺寸的移动设备

## 开始使用

### 前置要求
- 微信开发者工具
- 微信小程序账号

### 安装步骤

1. 克隆项目
```bash
git clone https://github.com/你的用户名/你的仓库名.git
```

2. 使用微信开发者工具打开项目

3. 配置AppID
   - 在微信开发者工具中设置你的小程序AppID

4. 编译运行

### 配置说明

如果需要使用食物识别功能，需要配置扣子AI：

1. 复制 `config/coze-config.example.js` 为 `config/coze-config.js`
2. 填入你的扣子AI配置信息

## 主要页面

- **首页**：展示今日目标、快速开始、推荐训练、本周数据
- **训练页面**：浏览各种训练计划
- **自定义计划**：创建和管理个人训练计划
- **训练执行**：跟随训练计划进行锻炼
- **肌肉分析**：学习肌肉知识和计算肌肉张力
- **食物识别**：拍照识别食物营养信息
- **训练记录**：查看历史训练数据

## 开发说明

### 添加新的训练动作

在 `pages/training-session/training-session.js` 中添加新的训练计划和动作。

### 添加新的SVG图标

在 `images/` 目录下创建新的SVG文件，建议尺寸为 100x100。

### 数据存储

项目使用微信小程序的本地存储API：
- `trainingRecords`: 训练记录
- `userInfo`: 用户信息
- `customPlans`: 自定义计划
