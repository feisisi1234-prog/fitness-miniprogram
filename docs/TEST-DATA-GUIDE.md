# 测试数据生成指南

## 📋 功能说明

为了方便测试小程序的各项功能，我们提供了测试数据生成工具，可以快速生成2月份10天的训练记录。

## 🎯 使用方法

### 方法1：通过设置页面（推荐）

1. 打开小程序
2. 点击底部导航栏的"我的"
3. 点击右上角的"设置"图标（⚙️）
4. 在"数据管理"区域找到以下功能：
   - 🧪 **生成测试数据** - 生成2月份10天的训练记录
   - 📊 **查看所有记录** - 查看当前所有训练记录统计
   - 🗑️ **清除所有记录** - 清空所有训练记录

### 方法2：直接调用工具函数

在任意页面的 JS 文件中：

```javascript
// 引入工具
const testDataUtils = require('../../utils/generate-test-data.js');

// 生成测试数据
testDataUtils.generateTestData();

// 查看所有记录
testDataUtils.viewAllRecords();

// 清除所有记录
testDataUtils.clearAllRecords();
```

## 📊 生成的测试数据

### 数据范围
- **日期**：2026年2月的10个随机日期
  - 2月3日、5日、8日、10日、12日、15日、18日、20日、23日、25日
- **时间**：每天8:00-20:00之间的随机时间
- **训练类型**：随机选择以下之一
  - 全身力量训练
  - HIIT燃脂训练
  - 瑜伽拉伸
  - 腹肌撕裂者

### 数据详情

每条记录包含：
- ✅ 训练计划ID和名称
- ✅ 训练类型（力量训练/有氧运动/柔韧性）
- ✅ 训练日期和时间
- ✅ 训练时长（秒）
- ✅ 格式化时长（如"25分钟30秒"）
- ✅ 消耗卡路里
- ✅ 完成的动作数
- ✅ 训练进度（70-100%）
- ✅ 时间戳

## ⚠️ 注意事项

1. **数据覆盖**：
   - 生成测试数据时，会自动删除2026年2月的旧数据
   - 其他月份的数据不受影响

2. **数据持久化**：
   - 测试数据保存在本地存储中
   - 清除缓存不会删除训练记录
   - 只有"清除所有记录"或"重置数据"会删除

3. **数据限制**：
   - 系统只保留最近30天的记录
   - 超过30天的记录会被自动清理

4. **连续天数计算**：
   - 连续天数从最近的训练日期开始往前计算
   - 如果今天没有训练，会从昨天开始计算
   - 中断一天就会重新计数

如果需要自定义测试数据，可以修改 `utils/generate-test-data.js` 文件：

### 修改日期范围
```javascript
const testDates = [
  '2026-02-03',
  '2026-02-05',
  // 添加更多日期...
];
```

### 修改训练计划
```javascript
const trainingPlans = [
  {
    id: 1,
    typeName: '自定义训练',
    type: '力量训练',
    durationRange: [1200, 2100],
    caloriesRange: [150, 250]

  // 添加更多计划...
];
```

```javascript
// 当前是8:00-20:00，可以修改为其他时间段
const hour = Math.floor(Math.random() * 12) + 8; // 8-19点
```
