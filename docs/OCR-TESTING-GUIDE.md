## 🐛 常见问题排查

### 问题1：识别质量评分低于60分

**可能原因**：
- 图片质量差
- 文字不清晰
- 格式不标准
- OCR API失败

**排查步骤**：
1. 查看原始文字，确认OCR是否正确识别
2. 检查图片是否清晰
3. 检查格式是否符合要求
4. 查看控制台日志

### 问题2：动作名称识别为"动作1"、"动作2"

**可能原因**：
- 动作名称不在识别库中
- OCR识别错误
- 格式不标准

**排查步骤**：
1. 查看原始文字，确认OCR识别的动作名称
2. 检查动作名称是否在支持列表中
3. 尝试使用标准动作名称
4. 查看控制台日志中的匹配过程

### 问题3：组数次数识别错误

**可能原因**：
- 数字格式不标准
- 多个数字混淆
- OCR识别错误

**排查步骤**：
1. 查看原始文字，确认OCR识别的数字
2. 检查数字格式是否清晰
3. 尝试使用标准格式：3组 x 12次
4. 查看控制台日志中的数字提取过程

### 问题4：休息时间使用默认值

**可能原因**：
- 没有明确标注休息时间
- 格式不标准
- OCR识别错误

**排查步骤**：
1. 查看原始文字，确认是否包含休息时间
2. 检查休息时间格式：休息60秒 或 60s
3. 如果没有标注，系统会使用智能默认值

### 问题5：完全识别失败

**可能原因**：
- 图片质量太差
- OCR API不可用
- 格式完全不标准

**排查步骤**：
1. 检查是否在真机上测试（开发工具不支持OCR API）
2. 查看控制台是否有错误信息
3. 尝试使用手动输入功能
4. 尝试使用模拟数据

## 💡 优化建议

### 对于用户

1. **使用标准格式**：动作名 3组 x 12次 休息60秒
2. **确保文字清晰**：字体大小14-18号，黑色文字，白色背景
3. **每行一个动作**：避免多个动作在同一行
4. **使用打印文字**：避免手写，手写识别率较低
5. **光线充足**：拍照时确保光线充足，避免阴影

### 对于开发者

1. **扩展动作库**：添加更多动作变体和别名
2. **优化解析算法**：改进数字提取和格式识别
3. **添加机器学习**：使用AI模型提高识别准确度
4. **用户反馈**：收集用户反馈，持续改进
5. **A/B测试**：测试不同的解析策略
