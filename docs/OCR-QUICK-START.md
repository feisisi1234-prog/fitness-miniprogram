# OCR识别快速入门

## 🚀 5分钟上手指南

### 第1步：准备图片

**最佳格式**：
```
全身力量训练

深蹲 3组 x 12次 休息60秒
俯卧撑 3组 x 10次 休息60秒
平板支撑 3组 x 30秒 休息45秒
弓步蹲 3组 x 10次 休息60秒
```

**简短标题格式**（v4.1新增）：
```
胸

坐姿下斜推 3组 × 12次 组内暂停 10+5 休息60秒
上斜推 3组 × 12次 休息60秒
飞鸟 3组 × 15次 休息45秒
```

**关键要点**：
- ✅ 每行一个动作
- ✅ 文字清晰，背景简洁
- ✅ 使用打印文字（不要手写）
- ✅ 光线充足
- ✅ 支持简短标题（胸、背、腿、肩等）
- ✅ 支持组内暂停格式（10+5）

### 第2步：扫描识别

1. 打开小程序
2. 进入"计划"页面
3. 点击"📷 扫描图片"
4. 阅读拍照提示
5. 选择图片或拍照
6. 等待识别完成

### 第3步：查看结果

识别完成后会显示：
- 计划名称
- 动作列表
- 组数、次数、休息时间
- 识别质量评分

**如果评分低于60分**：
- 查看问题和建议
- 点击"选项" → "查看识别建议"
- 根据建议改进图片
- 重新识别

### 第4步：保存或修改

**直接保存**：
- 点击"保存"按钮
- 计划自动添加到列表

**修改后保存**：
- 点击"选项"
- 选择"修改计划"
- 编辑动作信息
- 保存计划

## 🎯 支持的动作

### 下肢动作
深蹲、硬拉、弓步蹲、臀桥、腿举

### 胸部动作（v4.1新增）
坐姿下斜推、上斜推、平板推、飞鸟、绳索夹胸、卧推

### 背部动作（v4.1新增）
高位下拉、坐姿划船、杠铃划船、直臂下压

### 上肢动作
卧推、引体向上、俯卧撑、肩推、侧平举、二头弯举、三头屈伸

### 核心动作
平板支撑、仰卧起坐、卷腹、俄罗斯转体

### 有氧动作
开合跳、波比跳、高抬腿、登山跑、跳绳、跑步

### 瑜伽动作
猫牛式、下犬式、战士式、树式、婴儿式

**提示**：每个动作都支持多种叫法，例如：
- 深蹲 = 蹲起 = 下蹲 = squat
- 俯卧撑 = 伏地挺身 = push up
- 平板支撑 = 平板 = plank

## 🔧 常见问题

### Q1：识别不准确怎么办？

**A1**：
1. 点击"选项" → "查看原始文字"
2. 查看OCR识别的内容
3. 如果OCR正确但解析错误，点击"重新解析"
4. 如果OCR错误，改进图片后重新识别

### Q2：动作名称显示为"动作1"？

**A2**：
1. 检查动作名称是否在支持列表中
2. 使用标准动作名称
3. 点击"修改计划"手动修改

### Q3：组数次数不对？

**A3**：
1. 使用标准格式：3组 x 12次
2. 确保数字清晰可见
3. 点击"修改计划"手动修改

### Q4：在开发工具中无法识别？

**A4**：
- 开发工具不支持OCR API
- 会自动使用模拟识别
- 选择预设的训练计划
- 或使用"手动输入"功能

### Q5：如何提高识别成功率？

**A5**：
1. 使用标准格式
2. 确保文字清晰
3. 每行一个动作
4. 使用打印文字
5. 光线充足
6. 背景简洁

## 📊 识别质量评分

### 90-100分：优秀
- 识别非常准确
- 可以直接保存

### 70-89分：良好
- 识别基本准确
- 建议检查后保存

### 50-69分：一般
- 识别有些问题
- 建议修改后保存

### 0-49分：较差
- 识别失败
- 建议改进图片后重新识别
