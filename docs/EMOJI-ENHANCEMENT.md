# Emoji表情增强方案

## 🎨 设计理念

为了让界面更生动有趣，我们为所有使用占位图的动作添加了emoji表情。每个emoji都精心挑选，与动作特点相匹配。

| 动作 | Emoji | 含义 |
|------|-------|------|
| 仰卧起坐 | 💪 | 代表力量和肌肉训练 |
| 俄罗斯转体 | 🔄 | 代表旋转动作 |
| 平板支撑抬腿 | 🦵 | 代表腿部动作 |
| 卷腹 | ⚡ | 代表快速有力的动作 |

### 计划5：有氧舞蹈 💃

| 动作 | Emoji | 含义 |
|------|-------|------|
| 热身舞步 | 🎵 | 代表音乐和节奏 |
| 有氧舞蹈组合 | 💃 | 代表舞蹈动作 |
| 高强度舞步 | 🔥 | 代表燃脂和高强度 |
| 放松舞步 | ✨ | 代表放松和结束 |

| 动作 | Emoji | 含义 |
|------|-------|------|

| 侧卧抬腿 | 🦿 | 代表腿部动作 |

## 🎯 设计原则

### 1. 直观性
每个emoji都能直观地表达动作特点：
- 💪 = 力量
- 🔄 = 旋转
- 🦵 = 腿部
- 🎵 = 音乐
- 🔥 = 燃脂

### 2. 一致性
同类动作使用相似的emoji：
- 腿部动作：🦵 🦿
- 旋转动作：🔄 ⭕
- 力量动作：💪 ⚡

### 3. 美观性
emoji颜色丰富，让界面更生动：
- 🔥 红色 - 高强度
- 💃 粉色 - 舞蹈
- 🌬️ 蓝色 - 呼吸
- ✨ 金色 - 放松

## 💡 使用技巧

### 1. 动作名称中添加emoji
```javascript
{
  name: '仰卧起坐 💪',
  // ...
}
```

### 2. 保持占位图
```javascript
{
  image: '/images/placeholder.png',
  // 占位图仍然显示，但有emoji增强
}
```

### 3. 描述保持专业
```javascript
{
  description: '仰卧，双手放于耳侧或胸前，利用腹肌力量坐起，避免颈部用力'
  // 描述不添加emoji，保持专业性
}
```
