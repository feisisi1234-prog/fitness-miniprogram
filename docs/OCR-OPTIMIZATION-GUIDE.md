### 改进内容

#### 1. 扩展动作库（v4.0增强）

**新增动作变体、错别字和简写**：
- 深蹲 → 深蹲、蹲起、下蹲、蹲、深蹲起、深蹲跳、squat、squats、深墩、深顿
- 俯卧撑 → 俯卧撑、伏地挺身、掌上压、俯卧、push up、pushup、push-up、俯握撑、伏卧撑
- 引体向上 → 引体向上、拉单杠、单杠、引体、pull up、pullup、chin up、引休向上
- 平板支撑 → 平板支撑、平板、支撑、plank、planks、平版支撑、平板之撑
- 等30+个动作的多种表达方式，包含常见错别字

#### 2. 多模式数字提取（v4.0增强）

**支持的格式**：
```
✅ 深蹲 3组 x 12次 休息60秒
✅ 深蹲 3x12 60s
✅ 深蹲 3组12次
✅ 深蹲 3*12 rest 60
✅ 深蹲 3 sets 12 reps
✅ Squat 3x12 60s
✅ 深蹲 3 12 60（纯数字）
```

**智能推断**：
- 如果只有2个数字，第一个是组数，第二个是次数
- 如果有3个数字，第三个是休息时间
- 数字范围验证：组数1-10，次数5-100，休息10-300秒

#### 3. 智能推断（v4.0增强）

- **4层动作名称匹配**：
  1. 精确匹配（整词匹配，优先级最高）
  2. 模糊匹配（包含即可）
  3. 提取行首中文词组（2-8个字，包含动作关键字）
  4. 提取英文词组（支持多词组合）

- **智能默认休息时间**：
  - 深蹲、硬拉、卧推、引体向上：90秒
  - 开合跳、波比跳、高抬腿、登山跑：30秒
  - 平板支撑、仰卧起坐、卷腹：45秒
  - 其他动作：60秒

- **文本预处理**：
  - 自动去除中文标点符号
  - 统一空格格式
  - 跳过标题行和无关内容

#### 4. 调试功能（v4.0增强）

- **识别质量评估**：

- **拍照前指导**：

- **调试选项**：

## 🛠️ 常见问题解决

### 问题1：动作名称识别错误

**症状**：动作名称显示为"动作1"、"动作2"

**原因**：
- 动作名称不在识别库中
- OCR识别错误
- 格式不标准

**解决方法**：
1. 使用标准动作名称
2. 查看原始文字，确认OCR是否正确
3. 手动修改动作名称

**支持的动作名称**：
```
下肢：深蹲、硬拉、弓步蹲、臀桥、腿举
上肢：卧推、引体向上、俯卧撑、肩推、侧平举、二头弯举、三头屈伸
核心：平板支撑、仰卧起坐、卷腹、俄罗斯转体
有氧：开合跳、波比跳、高抬腿、登山跑、跳绳、跑步
瑜伽：猫牛式、下犬式、战士式、树式、婴儿式
```

### 问题2：组数次数识别错误

**症状**：组数或次数不正确

**原因**：
- 数字格式不标准
- 多个数字混淆
- OCR识别错误

**解决方法**：
1. 使用标准格式：`3组 x 12次`
2. 确保数字清晰
3. 手动修改

**支持的格式**：
```
✅ 3组 x 12次
✅ 3x12
✅ 3*12
✅ 3 sets 12 reps
✅ 3組12次
```

### 问题3：休息时间识别错误

**症状**：休息时间不正确或使用默认值

**原因**：
- 没有明确标注休息时间
- 格式不标准

**解决方法**：
1. 明确标注：`休息60秒`
2. 使用简写：`60s`
3. 系统会根据动作类型设置默认值：
   - 深蹲、硬拉、卧推：90秒
   - 开合跳、波比跳：30秒
   - 其他动作：60秒

### 问题4：完全识别失败

**症状**：只显示"动作1"、"动作2"

**原因**：
- 图片质量太差
- 格式完全不标准
- OCR完全失败

**解决方法**：
1. 重新拍照，确保清晰
2. 使用"查看原始文字"查看OCR结果
3. 使用"手动输入"功能
4. 使用"修改"功能手动编辑
