# OCR图片扫描训练计划功能指南

## 📸 功能介绍

通过拍照或上传图片，自动识别图片中的训练计划文字，快速创建自定义训练计划。

## 🎯 使用方法

### 1. 进入计划页面

点击底部导航栏的"计划"标签

### 2. 点击扫描按钮

在页面顶部有两个按钮：
- **手动创建** - 传统的手动输入方式
- **📷 扫描图片** - 新增的OCR识别功能

### 3. 选择图片

点击"扫描图片"后，可以选择：
- 📷 拍照 - 直接拍摄训练计划图片
- 🖼️ 从相册选择 - 选择已有的图片

### 4. 自动识别

系统会自动识别图片中的文字，并解析出：
- 计划标题
- 训练类别（力量训练/有氧运动/柔韧性）
- 动作列表
- 组数、次数、休息时间

### 5. 确认或修改

识别完成后会显示结果：
- **保存** - 直接保存识别的计划
- **修改** - 打开编辑表单进行调整

### 6. 开始训练

保存后，点击"开始训练"即可进入训练会话页面

## 📝 支持的图片格式

### 推荐的图片内容格式

**格式1：标准格式**
```
全身力量训练计划
深蹲 3组 x 12次 休息60秒
俯卧撑 3组 x 10次 休息60秒
平板支撑 3组 x 30秒 休息45秒
```

**格式2：简化格式**
```
臀腿训练
深蹲 3x12 60s
硬拉 3x10 90s
弓步蹲 3x10 60s
```

**格式3：中文格式**
```
HIIT燃脂
开合跳 4组12次
高抬腿 4组12次
波比跳 3组10次
```

### 识别的关键信息

系统会自动识别以下内容：

1. **计划标题**
   - 包含"训练"、"计划"等关键词
   - 或图片顶部的文字

2. **训练类别**
   - 力量训练：深蹲、卧推、硬拉、哑铃等
   - 有氧运动：跑步、跳绳、HIIT、开合跳等
   - 柔韧性：瑜伽、拉伸、放松等

3. **动作信息**
   - 动作名称：深蹲、俯卧撑等
   - 组数：3组、4组等
   - 次数/时长：12次、30秒等
   - 休息时间：60秒、90秒等

## 🎨 拍照技巧

### 最佳拍照条件

1. **光线充足**
   - 避免阴影遮挡文字
   - 自然光或明亮的室内光

2. **图片清晰**
   - 对焦准确
   - 避免模糊或抖动

3. **文字完整**
   - 确保所有文字都在画面内
   - 避免文字被遮挡或截断

4. **背景简洁**
   - 纯色背景最佳
   - 避免复杂的背景干扰

### 拍照示例

✅ **好的示例**：
- 白纸黑字，清晰可见
- 文字排列整齐
- 光线均匀

❌ **不好的示例**：
- 文字模糊不清
- 光线不足或过曝
- 文字倾斜或变形

## 🔍 识别的动作库

系统内置了常见的训练动作，可以自动识别：

### 力量训练动作
- 深蹲、硬拉、卧推
- 引体向上、俯卧撑、平板支撑
- 弓步蹲、臀桥、侧平举
- 肩推、二头弯举、三头屈伸
- 仰卧起坐、卷腹、俄罗斯转体
- 腿举、箭步蹲

### 有氧运动动作
- 开合跳、波比跳、高抬腿
- 登山跑、跳绳、跑步

### 柔韧性动作
- 猫牛式、下犬式、战士式
- 树式、婴儿式

## ⚙️ 高级功能

### 1. 编辑识别结果

如果识别不准确，可以：
- 点击"修改"按钮
- 在编辑表单中调整
- 修改动作名称、组数、次数等

### 2. 添加更多动作

识别后可以继续添加：
- 点击"添加动作"
- 从模板快速添加
- 或手动输入新动作

### 3. 上传动作图片

为每个动作添加标准图：
- 点击图片上传区域
- 选择动作示范图
- 帮助训练时参考

## 🚀 训练会话功能

点击"开始训练"后，进入训练会话页面：

### 页面功能

1. **计时器**
   - 自动计时训练时长
   - 显示当前动作用时

2. **卡路里消耗**
   - 实时计算消耗的卡路里
   - 根据训练强度自动估算

3. **动作信息**
   - 当前动作名称
   - 组数和次数
   - 动作描述和图片

4. **进度跟踪**
   - 当前第几组
   - 完成的动作数
   - 总体进度百分比

5. **休息提醒**
   - 组间休息倒计时
   - 语音提示（可选）

### 操作按钮

- ⏸️ **暂停/继续** - 暂停或继续训练
- ⏭️ **下一组** - 跳过当前组
- ⏭️ **下一个动作** - 跳到下一个动作
- ✅ **完成训练** - 结束并保存记录

## 📊 训练记录

完成训练后，系统会自动保存：
- 训练日期和时间
- 训练时长
- 消耗的卡路里
- 完成的动作数
- 训练进度

可以在"训练记录"页面查看历史记录。

## ❓ 常见问题

### Q1: 识别不准确怎么办？

**A**: 可以选择"修改"按钮，手动调整识别结果。建议：
- 确保图片清晰
- 文字排列整齐
- 使用标准格式

### Q2: 支持哪些语言？

**A**: 目前主要支持中文识别，包括：
- 简体中文
- 常见的训练术语
- 数字和单位

### Q3: 识别失败怎么办？

**A**: 如果识别失败：
1. 检查图片质量
2. 重新拍照或选择图片
3. 或使用"手动创建"功能

### Q4: 可以识别手写文字吗？

**A**: 可以，但建议：
- 字迹工整清晰
- 避免连笔
- 使用黑色笔书写

### Q5: 识别需要网络吗？

**A**: 
- 使用微信OCR API需要网络
- 识别速度取决于网络状况
- 建议在WiFi环境下使用
