# 视频域名配置指南

## 问题描述

在微信小程序中播放Supabase托管的视频时，可能会遇到以下错误：

```
Failed to load media https://jrchbzzwgqndaxsckvub.supabase.co/storage/v1/object/public/video/muscle.mp4
net::ERR_FAILED
```

这是因为微信小程序要求所有网络请求的域名必须在后台配置白名单。

## 解决方案

### 步骤1：登录微信公众平台

访问 [微信公众平台](https://mp.weixin.qq.com/)，使用小程序管理员账号登录。

### 步骤2：进入开发设置

导航路径：
```
开发 > 开发管理 > 开发设置 > 服务器域名
```

### 步骤3：配置域名

在 **request合法域名** 中添加：

```
https://jrchbzzwgqndaxsckvub.supabase.co
```

**注意事项：**
- 必须使用 `https://` 协议
- 域名必须已备案（Supabase的域名已备案）
- 每月只能修改5次，请谨慎操作
- 最多可配置20个域名

### 步骤4：保存并重新编译

1. 点击"保存并提交"
2. 在微信开发者工具中重新编译项目
3. 测试视频播放功能

## 开发阶段临时解决方案

如果暂时无法配置域名（例如没有管理员权限），可以在开发阶段使用以下方法：

### 方法1：开发者工具设置

在微信开发者工具中：
1. 点击右上角"详情"
2. 勾选"不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书"

**注意：** 此方法仅在开发阶段有效，真机预览和正式版本仍需配置域名。

### 方法2：真机调试

使用真机调试功能，可以绕过域名校验：
1. 点击工具栏的"真机调试"
2. 扫码在手机上打开
3. 在真机上测试视频功能

## 视频要求

确保视频符合微信小程序的要求：

- **格式：** MP4
- **编码：** H.264
- **分辨率：** 建议不超过1280x720
- **大小：** 建议不超过10MB
- **码率：** 建议500-2000kbps

## 验证配置

配置完成后，可以通过以下方式验证：

1. **开发者工具：** 关闭"不校验合法域名"选项，重新编译
2. **真机预览：** 使用预览功能在手机上测试
3. **体验版：** 上传体验版进行测试

## 常见问题

### Q1: 配置后仍然无法播放？

**解决方法：**
- 确认域名格式正确（包含https://）
- 重新编译项目
- 清除缓存后重试
- 检查视频URL是否正确

### Q2: 提示"域名格式不正确"？

**解决方法：**
- 确保使用完整的域名格式：`https://jrchbzzwgqndaxsckvub.supabase.co`
- 不要包含路径部分（如 `/storage/v1/...`）

### Q3: 开发者工具能播放，真机不能播放？

**解决方法：**
- 检查是否勾选了"不校验合法域名"
- 确认域名已在后台配置
- 等待配置生效（通常立即生效）

### Q4: 每月只能修改5次，用完了怎么办？

**解决方法：**
- 等待下个月重置
- 联系微信客服申请额外修改次数
- 使用已配置的域名托管视频

## 替代方案

如果无法配置Supabase域名，可以考虑：

1. **使用腾讯云COS：** 微信小程序推荐使用腾讯云服务
2. **使用微信云开发：** 自带域名白名单
3. **使用CDN：** 配置自己的域名并使用CDN加速
